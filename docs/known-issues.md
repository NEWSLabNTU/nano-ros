# Known Issues

> **Moved.** Issue tracking now lives in [`docs/issues/`](issues/) — one file
> per issue. See [`docs/issues/README.md`](issues/README.md) for the index
> and conventions.

The old single-file `#N` entries map to `docs/issues/NNNN-*` with the **same
numbers** (e.g. old `#7` → `docs/issues/0007-seq-capacity-64.md`). Open
issues live in `docs/issues/`; resolved ones in `docs/issues/archived/`.

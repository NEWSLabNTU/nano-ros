# C API

The C API reference is generated by Doxygen from `packages/core/nros-c/include/nros/*.h`
and hosted alongside this book at:

- [**`api/c/index.html`**](../api/c/index.html)

Always reflects the current `main` branch.

## Where to start

- [`nros/nros.h`](../api/c/nros_8h.html) — convenience umbrella include
- [`nros/init.h`](../api/c/init_8h.html) — `nros_support_init`, `nros_support_fini`
- [`nros/node.h`](../api/c/node_8h.html) — `nros_node_init`, `nros_node_fini`
- [`nros/executor.h`](../api/c/executor_8h.html) — spin loop, executor lifecycle
- [`nros/publisher.h`](../api/c/publisher_8h.html) / [`subscription.h`](../api/c/subscription_8h.html) — pub/sub
- [`nros/service.h`](../api/c/service_8h.html) / [`client.h`](../api/c/client_8h.html) — services
- [`nros/action.h`](../api/c/action_8h.html) — actions
- [`nros/parameter.h`](../api/c/parameter_8h.html) — parameter services
- [`nros/lifecycle.h`](../api/c/lifecycle_8h.html) — REP-2002 lifecycle states

## CMake integration

Out-of-tree projects use the `NanoRos::C` and `NanoRos::Cpp` CMake targets and
the `nano_ros_generate_interfaces()` function — covered in
[`docs/reference/c-api-cmake.md`](https://github.com/NEWSLabNTU/nano-ros/blob/main/docs/reference/c-api-cmake.md).

## Generating locally

```bash
doxygen packages/core/nros-c/Doxyfile
xdg-open target/doc/c-api/html/index.html
```

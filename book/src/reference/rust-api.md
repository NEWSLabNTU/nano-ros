# Rust API

The Rust API reference is generated from the source by `rustdoc` and
hosted alongside this book at:

- [**`api/rust/nros/`**](../api/rust/nros/index.html) — umbrella crate (most users)
- [`api/rust/nros_rmw/`](../api/rust/nros_rmw/index.html) — RMW trait surface
- [`api/rust/nros_platform_api/`](../api/rust/nros_platform_api/index.html) — platform abstraction traits
- [`api/rust/nros_rmw_zenoh/`](../api/rust/nros_rmw_zenoh/index.html) — zenoh-pico backend internals

Generated by `cargo doc --no-deps`. Always reflects the current `main`
branch — use the in-repo source if you need a specific commits docs.

## Where to start

- New to nano-ros? → [`nros::prelude`](../api/rust/nros/prelude/index.html).
- Writing an executor-driven app? → [`nros::Executor`](../api/rust/nros/struct.Executor.html).
- Custom platform port? → [`nros_platform_api`](../api/rust/nros_platform_api/index.html).

## Generating locally

```bash
cargo doc --no-deps -p nros -p nros-rmw -p nros-platform-api -p nros-rmw-zenoh --open
```

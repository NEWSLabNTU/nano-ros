# nros-tests/bins

Fixture binaries that `nros-tests` integration tests build and invoke
(e.g., QEMU test images).

These are NOT user-facing examples — see `examples/` at the repo root
for those. Each sub-directory is a standalone Cargo package.

Populated by Phase 131 Group B. See
`docs/roadmap/phase-131-examples-tree-revision.md`.

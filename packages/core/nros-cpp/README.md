# nros-cpp

C++ API for [nano-ros](https://github.com/NEWSLabNTU/nano-ros). Header-only C++14 wrapper on top of `nros-c`. Provides RAII handles, typed pub/sub, services, and futures while keeping the static-library footprint identical to the pure-C path.

See the [C++ API reference](https://github.com/NEWSLabNTU/nano-ros/tree/main/book/src/reference/cpp-api.md).

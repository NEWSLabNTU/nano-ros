# nros-c

C API for [nano-ros](https://github.com/NEWSLabNTU/nano-ros). Ships an rclc-compatible header surface plus a static library suitable for embedding from C / C++ applications on FreeRTOS, NuttX, Zephyr, ThreadX, or POSIX hosts.

See the [C API reference](https://github.com/NEWSLabNTU/nano-ros/tree/main/book/src/reference/c-api.md).

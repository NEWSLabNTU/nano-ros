# xrce-platform-shim

Forwarders from Micro-XRCE-DDS-Client C symbols to [nano-ros](https://github.com/NEWSLabNTU/nano-ros) `nros-platform` traits — the analogue of `zpico-platform-shim` for the XRCE backend.

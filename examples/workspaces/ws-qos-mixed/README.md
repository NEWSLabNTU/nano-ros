# ws-qos-mixed — C QoS node packages under a C++ typed entry

## What it shows

The **C** `qos_talker` / `qos_listener` node packages reused **verbatim** from
[`ws-qos-c`](../ws-qos-c/) (reliable + transient-local + keep-last-10 on
`/chatter`), driven by entries generated by the **C++ TYPED entry carrier**.
Two processes (issue 0096).

## Run

```sh
source ./activate.sh && nros ws sync
nros codegen-system --bringup demo_bringup
cmake -S . -B build && cmake --build build
zenohd --listen tcp/127.0.0.1:7447 &
./build/src/native_listener_entry/native_listener_entry &
./build/src/native_talker_entry/native_talker_entry
```

## Expected output

```
Published: 1        # talker
Received: 1         # listener
```
